# Release Checklist

## v0.1.0 MVP

1. Ensure all quality gates are green.
2. Run eval suite and review KPI summary.
3. Update release notes with:
   - value proposition
   - known limits (FastAPI+pytest only)
   - quickstart
4. Create tag: `v0.1.0`.
5. Publish GitHub Release.
6. Optionally publish package to PyPI.
